Paint Boy

The missing color inspector

Paint Boy is a tool for changing a div's color on the fly. Without Paint Boy, I found myself memorizing hex-colors and refreshing the page every time I wanted to change a div's color. Paint Boy simplifies this process by making it as easy as entering a css rule and selecting the background, border, or font color to change.

Try it out: Type a css rule into the color picker e.g. h1, p, #information-panel p, select background, border, or font color, and then change the color in the color-inspector.

I look forward to your feedback! I hope to improve Paint Boy in the future so that it's possible to manage a website's color palette.
