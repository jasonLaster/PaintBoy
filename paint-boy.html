Paint Boy

The missing color inspector

Paint Boy is the color inspector that is missing from webkit developer tools and firebug.

Paint Boy is perfect for tweaking a div's color on the fly. Stop memorizing hex-colors and tweaking colors one refresh at a time, thats just stupid.

If you like what you see, let me know. I have high hopes for Paint Boy becoming a comprehensive website color manager. I hope to build an eyedrop color selector and color palette viewer.

Try it out: Type in a css rule into the color viewer, select either the background, border, or font box and adjust the color in the color picker. e.g. h1, p, #information-panel p
